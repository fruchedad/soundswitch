## Detailed Description (16,000 characters max)

**Transform your SoundCloud experience with intelligent view switching and powerful analytics tools.**

SoundSwitch is the ultimate Chrome extension for SoundCloud power users, content creators, and anyone who wants to see how their profile appears to different audiences. With just one click, toggle between your authenticated view and the public view that visitors see.

### 🔄 **Smart View Switching**
- **Public View**: See exactly what anonymous visitors see on your profile
- **User View**: See your full authenticated profile with all private content
- **Instant Toggle**: Switch views without page reloads using our floating button
- **Profile Detection**: Automatically activates on SoundCloud profile pages

### 🎯 **Power User Features**
- **Auto-Hide Private Tracks**: Automatically collapse private tracks with toggle links
- **Hover Analytics**: View 7-day follower growth on hover over follower counts  
- **Quick Share**: Copy public profile URLs with one click
- **Dark Mode**: Beautiful dark theme that preserves images and videos
- **Real-time Updates**: All changes happen instantly without page refreshes

### 🛡️ **Privacy & Security**
When in Public View mode, SoundSwitch intelligently hides:
- Follower and following counts (your private metrics)
- Likes and reposts tabs (personal engagement data)
- Private tracks and playlists (content not meant for public)
- Comments sections (private interactions)
- Detailed track statistics (analytics data)

### 📊 **Analytics Dashboard**
- **Follower Growth Tracking**: Monitor new followers over the last 7 days
- **Growth Rate Calculation**: See percentage growth rates
- **Hover Stats**: Get insights without cluttering the interface
- **Cached Data**: Smart caching reduces API calls and improves performance

### 🎨 **Beautiful Design**
- **Non-intrusive Interface**: Floating button blends with SoundCloud's design
- **Dark Mode Support**: Elegant dark theme that works with all content
- **Responsive Controls**: Works perfectly on all screen sizes
- **Accessible Design**: Keyboard navigation and screen reader support

### ⚡ **Performance Optimized**
- **Lightweight**: Minimal impact on page loading times
- **Smart Caching**: 5-minute cache for analytics to reduce API calls
- **Memory Efficient**: Proper cleanup prevents memory leaks
- **Fast Switching**: Instant view changes without page reloads

### 🔧 **Easy Setup**
1. Install the extension
2. Visit any SoundCloud profile page
3. Look for the floating toggle button in the bottom right
4. Click to switch between Public and User views
5. Access more features via the extension popup

- **Privacy-Conscious Users**: Understand what information is public
- **Community Managers**: Verify profile presentation for brands

## Permissions Justification

### Required Permissions:
- **cookies**: Detect SoundCloud login status to provide appropriate view
- **activeTab**: Modify the current SoundCloud tab to switch views
- **storage**: Save user preferences and cache analytics data
- **scripting**: Inject view switching functionality into SoundCloud pages
- **alarms**: Schedule cache cleanup for optimal performance

### Host Permissions:
- **https://*.soundcloud.com/***: Access SoundCloud pages to provide functionality
- **https://api.soundcloud.com/***: Fetch user analytics for growth statistics
- **https://api-v2.soundcloud.com/***: Access enhanced SoundCloud API features

All permissions are used solely for the extension's core functionality and are never used to collect personal data or track user behavior.
