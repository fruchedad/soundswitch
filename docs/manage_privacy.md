# How to Manage Privacy for Your SoundCloud Profile and Content

## Profile Privacy

### Making Your Profile Private
1. Click your avatar → Settings
2. Navigate to Privacy tab
3. Enable "Private profile"
4. Confirm the change

### Effects of Private Profile:
- Only approved followers see your content
- New followers must request access
- Existing followers are preserved
- Public tracks remain accessible via direct link

## Content Privacy Settings

### Track Privacy Levels:
1. **Public** - Searchable and visible to all
2. **Private** - Only visible to you
3. **Unlisted** - Accessible via direct link only

### Setting Track Privacy:
- During upload: Select privacy before publishing
- After upload: Edit track → Basic info → Privacy

### Playlist Privacy:
- Public playlists visible on profile
- Private playlists hidden from profile
- Secret playlists shareable via link

## Managing Followers

### For Private Accounts:
- Review follower requests in notifications
- Approve or deny individually
- Remove followers from your follower list

### Blocking Users:
1. Visit user's profile
2. Click ••• menu
3. Select "Block user"

## Best Practices

- Regularly review privacy settings
- Use private uploads for works in progress
- Share secret links for feedback
- Consider playlist privacy for curated content
