# SoundCloud Account Privacy

## Private vs Public Accounts

### Public Account (Default)
- Profile visible to everyone
- Tracks, playlists, likes, and reposts are public
- Followers and following lists are visible
- Comments and stats are public

### Private Account
- Profile only visible to approved followers
- All content is private by default
- Must approve follower requests
- Existing followers retained when switching to private

## Privacy Settings

### What's Hidden in Private Mode:
- Track uploads (unless individually set to public)
- Playlists
- Likes and reposts
- Follower/following lists
- Profile stats

### What Remains Visible:
- Profile picture and header
- Display name
- Bio/description
- Public tracks (if any)

## Managing Privacy

1. Go to Settings > Privacy
2. Toggle "Make my profile private"
3. Manage follower requests
4. Set individual track privacy

## Track Privacy Options

- **Public**: Visible to everyone
- **Private**: Only you can see
- **Secret**: Shareable via secret link
